SIERPINSKI TRIANGLE
Difficulty: 3

------------------------------------------

PART I - Sierpinski.java

% javac Sierpinski.java
% java Sierpinski N
  > N = order of recursive depth (1-7 for best results)

Running this program recursively generates Sierpinski's triangle (a fractal graphic) of depth N.

------------------------------------------

PART II - Art.java

% javac Art.java
% java Art N multicolor
  > N = order of recursive depth (1-5 for best results)
  > multicolor = boolean true/false value for either black/white or multi-colored result

Running this program outputs a recursively-generated design of depth N made of circles.
