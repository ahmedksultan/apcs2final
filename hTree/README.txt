H TREE
Difficulty: 3

------------------------------------------

PART I - HTree.java

% javac HTree.java
% java HTree N
  > N = order of recursive depth (1-7 for best results)

Running this program draws an H tree of recursive depth N.

------------------------------------------

PART II - Art.java

% javac Art.java
% java Art N
  > N = order of recursive depth (1-5 for best results)

Running this program ouputs a recursively-generated design of depth N made of spots.
